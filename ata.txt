Monitor de umidade de solos para plantas domésticas – Arduino Uno

1. Ideia do projeto
O projeto consiste em um sistema de monitoramento da umidade de solos para plantas domésticas. Com a utilização do arduino e componentes eletrônicos, visamos que o projeto sirva como um indicador visual e sonoro do nível de umidade do solo. Neste projeto, temos três leds que acendem conforme o nível detectado pelo sensor de umidade: vermelho para solo seco, amarelo para umidade moderada e azul para solo totalmente úmido. Aĺém de informações visuais por led, contamos com a utilização do lcd 16x2 que disponibiliza informações mais detalhadas para o usuário e através de um teclado matricial deve também permitir opções de configurações. O sistema também deve ser capaz de emitir avisos sonoros conforme as configurações do usuário e armazenar hora e data para tais configurações.


2. Materiais utilizados
 - Arduino Uno
 - Cabo USB
 - Jumpers
 - Leds
 - Resistores
 - LCD 16x2
 - Teclado Matricial
 - Buzzer
 - Sensor de umidade
 - Potenciometro
 - RTC DS3231
